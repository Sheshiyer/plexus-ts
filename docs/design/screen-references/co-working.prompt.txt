This screen REPLACES the older meeting-oriented "Realtime" tab with an ambient co-working surface: presence-first, project-anchored, with a persistent voice lounge instead of scheduled meetings.

- Aesthetic: "The Operator — precision instrument for builders." Voice: precise, calm, technical. Bioluminescent calm.
- BANNED: Inter font, emoji icons, decorative color, outer neon glow, drop shadows, gradient text, frosted glass, real human faces (use abstract avatar circles with mono initials only).
